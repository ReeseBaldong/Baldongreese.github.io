Introduction

HTML, which stands for HyperText Markup Language is the standard language used to create and design web pages on the internet. HTML provides a structured way to organize content on a webpage by using systems of tags like "<>" symbols. These tags define various elements of the webpage such as headings, paragraphs, images, links, forms, and more. Each HTML tag serves a specific purpose and tells the web browser how to display the content in contains.

For example, a basic HTML document typically starts with an opening "html" tag, followed by a "head" section containing metadata like the page title and links to external sources such as stylesheets or scripts. A "body" section where the main context of the webpage resides. Within the body section, you can include various tags to structure you content like:

  • "h1" for headings
  • "p" for paragraphs
  • "img" for images
  • "a" for links
  • "ul" for unordered lists
  • "form" for input forms
HTML is not a programming language but rather a markup language, meaning it focuses on defining the structure and layout of a webpage rather than implementing logic or algorithms. It acts as the foundation upon which web browsers like Google Chrome, Mozilla Firefox, or Safari to render and display the content to users. When you create an HTML document, you're essentially providing instructions to the browser on how to present the information to visitors, ensuring consistency and compatibility across different platforms and devices.

CSS

CSS which stands for Cascading Stylesheet is another crucial component in web development, often used in conjunction with HTML. While HTML provides the structure and content of a webpage, CSS enhances its appearance and layout by applying styles and formatting.

CSS operates on a rule-based system, where each rule consists of a selector and a declaration block. The selector identifies which HTML elements the rule implies to, while the declaration block contains one or more property value pairs separated by semicolons. CSS offers a wide range of selectors, allowing developers to target elements based on their type, class, ID, attributes, or even their relationship with other elements in the HTML document. CSS also supports inheritance and cascading, which are fundamental concepts that styles applied to parent elements are automatically inherited by their child elements unless explicitly overridden.

Starting point
  • To get started with HTML and CSS, you'll need a text editor to write your code and a web browser to preview your webpages. You can choose from Visual Studio Code, Sublime Text or any other code editor of your choice
  • launch you text editor and create a new file and save it with a ".html" extension. for example, you could name it "index.html"
  • Inside your HTML file, you can start writing your HTML code to structure the content of your webpage. You can begin with changing the title or putting some content int the body like "h1" or "p" tags to start your sentences or phrases.
  • If you want to add CSS styling to your HTML document, create a enw file in your text editor and save it with a ".css" extension. for example, you could name it "style.css"
  • Link your HTML file to your CSS file using the "link" element inside the "head" section.
